## Project Description
This project is a simple Student Result Management System developed using Python. 
It allows users to add and view students' results.

## SDLC Model Used
Waterfall Model

## SDLC Explanation

### 1. Planning
The project was planned to help manage students' results easily.

### 2. Requirement Analysis
- Add student result
- View stored results
- Save data to a file

### 3. System Design
The system is divided into:
- main.py (controls the program)
- result_manager.py (handles logic)
- results.txt (stores data)

### 4. Implementation
The system was implemented using Python programming language.

### 5. Testing
The system was tested by adding and viewing multiple student results.

### 6. Deployment
The project was deployed by uploading it to GitHub.

### 7. Maintenance
Future improvements may include editing and deleting results.
